# **Portfolio v1.0 — Rizki R.**

A high-performance, minimalist portfolio designed for a Backend Engineer. Built with a focus on **Semantic HTML**, **Clean Code**, and **System Logic Storytelling**.

---

![desktop image](/assets/Desktop.png)

---

## **Project Goals**

- Menampilkan keahlian Backend (Go, Python, AI) melalui pendekatan "Logic First".
- Standar kualitas visual Awwwards dengan performa ringan (Zero-dependency).

## **Technical Stack**

- **Frontend:** HTML5 (Strictly Semantic), CSS3 (Modern Flex/Grid).
- **Typography:** Plus Jakarta Sans (Headers), Inter (Body), JetBrains Mono (Code).
- **Design System:** Minimalist Slate Navy & Ghost White.
- **Performance:** 100/100 Lighthouse Score (Targeted), no external libraries/frameworks.

## **Key Features**

- **Interactive API Status:** Menampilkan ketersediaan dan tech stack dalam format JSON yang interaktif.
- **The "Logic Breakdown":** Section khusus di setiap proyek yang menjelaskan solusi teknis atas masalah sulit (bukan sekadar daftar fitur).
- **Smooth Scroll Engine:** Implementasi pergerakan halaman yang halus dengan CSS cubic-bezier untuk sensasi inersia premium.
- **Responsive Sticky Navigation:** Navigasi cerdas dengan scroll-margin-top untuk UX yang mulus di mobile maupun desktop.
